🌐 Projeto de Interface – Mapa da Jornada
Este repositório reúne meu Mapa de Jornada Pessoal e Profissional e os protótipos de interface (UI) desenvolvidos durante as práticas. O objetivo é documentar minha evolução em Ciência da Computação, conectando valores pessoais, objetivos profissionais e design de interfaces acessíveis.

📌 Sobre o Projeto
Valores: Inovação tecnológica, Impacto social e econômico, Responsabilidade, Honestidade, Respeito, Empatia
Paixões: Tecnologia, Escrita, Leitura
Habilidades: Lógica computacional, Liderança, Consistência, Aprendizado contínuo
Objetivos: Especialização em tecnologia, oportunidades internacionais, qualidade de vida e conhecimento

📱 Protótipo SkillPath
Fluxo simples: Cadastro → Objetivos → Trilha de aprendizagem
Paleta roxa (criatividade e confiança), tipografia Inter, botões consistentes
Acessibilidade: contraste adequado, botões grandes, textos claros, compatibilidade futura com leitores de tela

📖 Reflexão
Aprendi: clareza de valores e diagnóstico de competências
Preciso ajustar: indicadores mais objetivos, prática semanal e networking ativo
Como me sinto: confiante e motivada, consciente das inseguranças, mas determinada a superá-las
